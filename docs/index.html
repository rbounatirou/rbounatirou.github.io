Exemples de projets personnels

Dans mon parcours d'autodidacte je me suis formé par de petits projets que j'ai pu réaliser. Voici une liste de quelques uns d'entre eux :

Description brève du projet Apports de ce projet Informations complémentaires
Création d'un jeu d'échec
  • Approfondissement des connaissances du langage JAVA
  • Initiation à la bibliothèque JavaFX
  • Utilisation de design patterns
 Ce qui m'a motivé dans ce projet a été la création d'un jeu relativement simple en java, afin d'apprendre à utiliser JavaFX et de reprendre notamment les concepts de thread. J'ai également pu utiliser de nouvelles connaissances avec les design patterns, notamment le pattern command
Création d'un petit jeu mêlant jeu de cartes et tactical-rpg
  • Approfondissement des connaissances sur le langage C++
  • Principale bibliothèque SFML 2.X
  • Initiation aux threads
 Lors de ce projet je voulais créer un jeu en m'inspirant de Hearthstone et de Dofus, en cherchant à mixer les jeux de cartes à jouer et de stratégie.
J'ai pu approfondir quelques notions et notamment débuter mon apprentissage de la programmation multi-threads.
Création d'une TODO list en ligne
  • Initiation au Javascript (Ajax)
  • Approfondissement du PHP
  • Approfondissement du HTML
  • Approfondissement du CSS
  • Approfondissement du SQL (SGBDR = mysql)

Dans ce projet je voulais créer un système de gestion mutualisé de TODO list basée sur des arborescences, dans lequel chaque personne pouvait recevoir un code unique pour accéder à une branche du projet dans le cas où elle avait été partagée par le créateur de la liste.

J'étais intéressé par l'approfondissement de mes connaisssances web lors de ce projet. Bien que non publié, j'ai pu aller au bout de ce projet.

Création d'une application mobile de suivi nutritionnel
  • Approfondissement du langage Java
  • Première expérience avec Android Studio
  • Première utilisation d'API externe (Open food facts)
  • Utilisation de Merise
  • Approfondissement du langage SQL (SGBDR= mysql)

Dans ce projet j'ai voulu manipuler android, en scannant des codes barres, puis grâce à ces derniers le programme interrogeait une API externe, afin d'y extraire les valeurs nutritionnelles et permettre de comptabiliser le total de ces dernières de manière journalière.
