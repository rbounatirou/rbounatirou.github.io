Exemples de projets personnels

Dans mon parcours d'autodidacte je me suis formé par de petits projets que j'ai pu réaliser. Voici une liste de quelques uns d'entre eux :

(Algorithmes génétiques)Projet pour rechercher des zones d'intérêts dans une ville + (Projet d'équipe) Les 8 reines

Apports personnels

  • Découverte des algorithmes génétiques
  • Mise en application des algorithmes génériques
  • Approfondissement C# + Winform
  • Approfondissement Git et Github
  • Travail d'équipe

Résumé du projet

Par curiosité et afin d'apprendre des notions vis à vis des algorithmes génétiques, j'ai préparé un projet dont le principe est basé sur une ville constituée de lieux (de plusieurs types), l'algorithme tente en fonction de critéres indiqués par l'utilisateur (interface winform), de trouver les points rassemblant le plus de critères de proximité avec les lieux désirés. Ce projet utilise également un design pattern visiteur.
Une fois ce projet terminé, et pour continuer les algorithmes génétiques, j'ai fait avec un collègue, un projet de puzzle des 8 reines Ce projet consiste à placer sur l'échiquier 8 reines, sans qu'aucune d'elles ne gêne le déplacement des autres. Au cours de ce projet j'ai pu approfondir mes compétences en communication, et mon utilisation de Git et Github en équipe.
